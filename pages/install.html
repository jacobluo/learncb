03操作步骤 — 准备工作

3.1

下载/安装 CodeBuddy < 5 分钟

1
访问官网

打开浏览器访问 https://www.codebuddy.cn，点击页面上的下载按钮。

2
选择平台并安装
🍎 macOS

下载 .dmg 文件，打开后拖入 Applications

🪟 Windows

下载 .exe 文件，双击运行安装向导

🐧 Linux

选择 .deb / .rpm 或 .tar.gz

3
登录账号

启动 CodeBuddy，按提示完成腾讯云账号登录/授权即可开始使用。

⚠️ 注意：请确保从官方渠道下载，保证安全性。
