06常见问题 FAQ

规则修改后不生效怎么办？

规则只在会话开始时加载。修改规则后，需要新建一个对话会话才能生效，当前会话不会自动刷新。

Memory 记忆功能如何管理？

在对话中说 "请记住…"、"删除…的记忆"、"更新记忆…" 即可。Memory 是全局的，在所有项目和对话中生效。建议保持 10-20 个以内。

MCP 安装失败怎么排查？

① 检查网络连接 ② 确认 command 路径正确（如 npx、python 是否在 PATH 中）③ 查看 MCP 状态指示灯，红色表示失败 ④ 尝试在终端手动运行 command 检查报错信息

三种模式如何选择？

Ask：纯问答不改代码 → Craft：单文件/小范围修改 → Plan：跨文件复杂任务。日常最常用 Craft，复杂需求用 Plan。

代码补全不工作？

① 确认已登录腾讯云账号 ② 检查网络连接 ③ 确认设置中代码补全功能已开启 ④ 尝试 Ctrl + Shift + Enter 手动触发

Rules 和 Memory 有什么区别？

Rules 是手动创建的行为规范（如"代码必须加注释"），需要配置文件；Memory 是 AI 自动记住的偏好和事实（如"我喜欢用 TypeScript"），通过对话创建。Rules 适合团队规范，Memory 适合个人偏好。
