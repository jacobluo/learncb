04开始 Coding

4.1 模式选择建议

简单问答 / API 查询
→ Ask 模式
例："React 18 中 useSyncExternalStore 怎么用？"
函数编写 / 小范围修改
→ Craft 模式
例："给这个组件添加 loading 状态和错误处理"
跨文件重构 / 复杂功能
→ Plan 模式
例："将整个项目从 JavaScript 迁移到 TypeScript"

4.2 分步骤操作指引

Step 1 打开项目

启动 CodeBuddy → File > Open Folder 打开项目目录
或终端执行：codebuddy /path/to/project

Step 2 打开对话面板

macOS：Ctrl + ⌘ + I ｜ Windows：Ctrl + Win + I

Step 3 选择交互模式

在对话面板顶部切换 Ask / Craft / Plan 模式

Step 4 描述需求（自然语言） 
# Craft 模式示例
帮我创建一个 Express 服务器，包含：
1. 用户注册接口 POST /api/register
2. 用户登录接口 POST /api/login
3. JWT 认证中间件
4. 用户信息查询 GET /api/profile（需认证）

技术要求：TypeScript + Zod 参数校验 + bcrypt 密码加密
Step 5 审阅并确认变更

在 Diff 视图中审阅 AI 生成的代码变更，点击 ✅ 接受或 ❌ 拒绝

Step 6 使用 @ 引用增强上下文
@文件名 将文件加入上下文 @文件夹 将目录加入上下文 @规则名 引用手动规则 @技能名 触发 Skill
Step 7 善用代码补全（NES）

Tab 接受补全建议 ｜ Ctrl + Shift + Enter 手动触发前瞻编辑
